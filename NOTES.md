# Teaching Notes

## User Preferences
- Wants to learn nginx comprehensively (all use cases)
- Complete beginner, but motivated
- Prefers understanding over copy-paste
- Wants practical results quickly

## Teaching Strategy
- Start with "what is nginx?" and architecture basics
- Follow with hands-on: serve a static site
- Then build toward more complex configurations
- Always tie lessons back to the mission

## Resources to Populate
- Official nginx documentation
- High-quality tutorials and guides
- Architecture diagrams and comparisons (nginx vs Apache, etc.)
