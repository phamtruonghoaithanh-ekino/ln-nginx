# Learning Resources

## Official Documentation
- **nginx Docs**: https://nginx.org/en/docs/
  - The authoritative reference for all nginx configuration and behavior
- **nginx Beginner's Guide**: https://nginx.org/en/docs/beginners_guide.html
  - Official tutorial covering static serving, reverse proxy, and FastCGI
- **HTTP proxy module docs**: https://nginx.org/en/docs/http/ngx_http_proxy_module.html
  - Full reference for `proxy_pass` and related directives
- **How nginx processes a request**: https://nginx.org/en/docs/http/request_processing.html
  - Server name resolution and location matching rules

## Architecture & Design
- **The Architecture of Open Source Applications — nginx**: https://aosabook.org/en/v2/nginx.html
  - In-depth chapter on nginx internals, worker model, and design philosophy
